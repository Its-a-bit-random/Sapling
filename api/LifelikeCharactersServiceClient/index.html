LifelikeCharactersServiceClient

This item only works when running on the client. Client

Can be accessed via LifelikeCharactersService.getClient.

Functions​

setHeadRotationReplicationRate​

</>
LifelikeCharactersServiceClient.setHeadRotationReplicationRate(rate: number) → ()

Override the head replication rate, by default the rate is 1/20.

WARNING

You shouldn't mess with this too much as higher values can cause bandwidth issues.

setFirstPersonOffset​

</>
LifelikeCharactersServiceClient.setFirstPersonOffset(offset: number) → ()

Sets how far farward the camera will be in first person compared to normal. Note that the negative scale is farward relative to the head.

updateSoundForMaterial​

</>
LifelikeCharactersServiceClient.updateSoundForMaterial(
material: Material,
sound: string?
) → ()

Sets the sound for the passed material to be sound where if sound is nil then the fallback sound is used, this by default is the normal roblox walking sound however this can be overrided with LifelikeCharactersServiceClient.changeFallbackSound.
